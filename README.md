# Tools and Tool Calling in LangChain

This repository demonstrates how to use **Tools** and **Tool Calling** in LangChain, enabling Large Language Models (LLMs) to interact with external functions and perform actions beyond text generation.

## Project Structure

```text
Tools-and-Tool-Calling-in-LangChain/
├── tools_in_langchain.ipynb
├── tool_calling_in_langchain.ipynb
├── README.md
├── LICENSE
└── .gitignore
```

## Features

- Creating Custom Tools
- Using Built-in LangChain Tools
- Tool Binding
- Tool Calling with LLMs
- Function Calling Workflows
- Practical Notebook Examples

## Prerequisites

- Python 3.10+
- LangChain
- langchain-core
- python-dotenv

## Installation

Clone the repository:

```bash
git clone https://github.com/ashislife/Tools-and-Tool-Calling-in-LangChain-.git
```

Navigate to the project directory:

```bash
cd Tools-and-Tool-Calling-in-LangChain-
```

Install the required dependencies:

```bash
pip install langchain langchain-core python-dotenv jupyter
```

## Usage

Start Jupyter Notebook:

```bash
jupyter notebook
```

Open and run:

- `tools_in_langchain.ipynb`
- `tool_calling_in_langchain.ipynb`

Execute the notebook cells sequentially.

## Files

| File | Description |
|------|-------------|
| `tools_in_langchain.ipynb` | Demonstrates creating and using tools in LangChain. |
| `tool_calling_in_langchain.ipynb` | Demonstrates tool calling workflows with LangChain and LLMs. |
